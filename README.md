# Free ID
## Description

Package to get a free id (for example: in XML)

## Installation

```composer require vitkuz573/free-id```

## Usage

```php
<?php

use Vitkuz573\FreeId;

// FreeId\inXml('test.xml', 'node', 'other_id', '10');
FreeId\inXml('test.xml', 'node');
```
