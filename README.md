# Get Free ID
## Description

Package to get a free id (for example: in XML)

## Requirements

- PHP >= 7.3

## Installation

```composer require vitkuz573/get-free-id```

## Usage

```php
<?php

use vitkuz573\GetFreeID;

// GetFreeId::inXML('test.xml', 'node', 'other_id', '10');
GetFreeID::inXML('test.xml', 'node');
```
